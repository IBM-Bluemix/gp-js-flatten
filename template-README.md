Usage
===

Given `input=`:

```json
{
    "foo": "foo",
    "bar": "bar",
    "baz": {
        "quux": true,
        "hey": "hi",
        "π": 3.14159,
        "ignored1": [],
        "ignored2": {}
    }
}
```

and

```js
const flatten = require('gp-flatten');
const flat = flatten.flatten(input);
```

gives

```json
{
    "$.foo": "foo",
    "$.bar": "bar",
    "$.baz.quux": "true",
    "$.baz.hey": "hi",
    "$.baz[\"π\"]": "3.14159"
}
```

and then

```js
const expand = flatten.expand(flat);
```

gives


```json
{
    "foo": "foo",
    "bar": "bar",
    "baz": {
        "quux": "true",
        "hey": "hi",
        "π": "3.14159"
    }
}
```
